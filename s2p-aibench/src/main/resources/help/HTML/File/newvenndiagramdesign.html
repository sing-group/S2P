New Venn Diagram design

This operation allows you to create a new Venn Diagram design.

Usage

You can execute this operation by clicking the menu option File/New Venn Diagram design. Since this operation does not require any parameter, an empty design will be added to the clipboard and the Venn Diagram designer will be automatically opened.

After clicking the Ok button, a new Venn Diagram design datatype is added to the clipboard and automatically opened in the Venn Diagram designer.
