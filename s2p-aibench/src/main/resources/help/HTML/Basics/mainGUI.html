Main user interface

The main user interface of S2P is organized into three main sections: the toolbar, the clipboard, and the analysis viewer. In the following figure, you can see the general aspect of S2P and the location of each section.

S2P main window
(click in the image to enlarge)

Through the toolbar and the menu bar, you will be able to access the main functions of S2P. On the clipboard tree, you will find a list with the data loaded. Finally, you will be able to explore your data through the different views of the data loaded shown in the view area.
