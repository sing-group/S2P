Fill plate

This operation allows you to distribute spots data into one or several Maldi plates..

Usage

You can execute this operation by clicking the menu option Operations/Fill plate.

A dialog will appear allowing you to specify the design of the Maldi plates and the spots data. As you can see in the following image, this dialog requires you to provide the following information:

  • Rows: the number of matrix rows.
  • Rows type: the type of rows (i.e. numeric or letters).
  • Columns: the number of matrix columns.
  • Columns type: the type of columns (i.e. numeric or letters).
  • Calibrants: whether calibrants must be added to the matrix or not.
  • Spots data: loaded spots data from the clipboard.
  • Replicates: the number of spot replicates that must be included in the Maldi plate.
  • Random: whether spots must be distributed randomly or not.
