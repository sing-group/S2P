Venn Diagram design

Venn Diagram designs can be edited using the Venn Diagram designer.

Venn Diagram designer

Briefly, the Venn Diagram designer allows you to define one or more sets and obtain the R code that creates the Venn Diagram.

Venn Diagram designer
(click in the image to enlarge)

The toolbar in the upper part of the editor has the following functions:

  • : allows you to select the target R library.
  • : adds a new set.
  • : generates the R code using the current configuration for the sets introduced.
  • : allows you to save the Venn Diagram design so that it can be loaded in S2P later using the Load Venn Diagram design operation.
