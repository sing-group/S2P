Mascot identifications

Mascot identifications can be explored and edited using the Mascot identifications table.

Mascot identifications table

The Mascot identifications table allows you to:

  • View, explore and sort the list of Mascot identifications loaded.
  • Manually delete Mascot identifications (selecting one or more rows and doing right-click).
  • Save the clean list of Mascot identifications as CSV for further uses in S2P or external applications.
  • View a summary of each column by hovering the mouse over the corresponding header.

To export or save the table you must click the button, which will show you the Export to CSV option that allows you to save the list of identifications into a CSV file.
