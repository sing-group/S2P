Load Venn Diagram design

This operation allows you to a Venn Diagram design from a file.

Venn Diagram design file

This option allows you to a Venn Diagram design from a file. This file can be created using the Venn Diagram designer to store the Venn Diagram design.

Usage

You can execute this operation by clicking the menu option Load/Load Venn Diagram design. A dialog will appear allowing you to choose the file containing the Venn Diagram design.

Load Venn Diagram design dialog

After clicking the Ok button, the loaded design is added as a Venn Diagram design datatype to the clipboard and automatically opened in the Venn Diagram designer.
